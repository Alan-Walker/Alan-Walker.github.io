如何使用 Hexo 创建自己的博客

安装 Hexo

所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

Hexo 建站

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

创建文章

  • 新建一篇文章

在 /source/_post 里添加 hello-world.md 文件，之后新建的文章都将存放在此目录下。
文章使用的是MarkDown 语法，编辑之后保存

  • 生成本地静态网站
  • 启动Hexo服务器

现在在浏览器输入 http://localhost:4000 ，可以看到生成的静态网站，使用的主题是默认的 landscape

关联Github

到目前为止，网站只能在本地访问，要想发布到互联网，可以使用Github作为仓库

  • 创建版本库
    要使用github作为服务器，首先去 github 创建一个新的版本库，Repository name 必须为 yourname.github.io (yourname与你的注册用户名一致,这个就是你博客的域名)

  • 关联Hexo
    在 _config.yml 文件中配置github的相关信息（yourname改为你的github用户名）

  • 安装Git 部署插件

  • 发布到Github

执行部署命令的时候会要求你输入github的账户名称和密码（透明的）。
稍等片刻，就可以使用通过 http://yourname.github.io 你的部署的网站。
